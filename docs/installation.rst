============
Installation
============

Stable release
--------------

To install Chariot Southbound Dispatcher, run this command in your terminal:

.. code-block:: console

    $ pip install chariot_southbound_dispatcher

This is the preferred method to install Chariot Southbound Dispatcher, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/

From sources
------------

The sources for Chariot Southbound Dispatcher can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git@gitlab.com:chariot-h2020/chariot-southbound-dispatcher.git

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://gitlab.com/chariot-h2020/chariot-southbound-dispatcher/-/archive/master/chariot-southbound-dispatcher-master.zip

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: git@gitlab.com:chariot-h2020/chariot-southbound-dispatcher.git
.. _tarball: https://gitlab.com/chariot-h2020/chariot-southbound-dispatcher/-/archive/master/chariot-southbound-dispatcher-master.zip
